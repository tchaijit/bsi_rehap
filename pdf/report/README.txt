วิธีใช้งาน Dashboard (มี 2 วิธี):

========================================
วิธีที่ 1: เปิดแบบ Static (แนะนำ - ง่ายที่สุด)
========================================

1. เปิดไฟล์ dashboard.html โดยตรง
   - ดับเบิลคลิก: dashboard.html
   - หรือคลิกขวา > Open with > Browser

2. Dashboard จะโหลดข้อมูลจาก data.js อัตโนมัติ

3. ข้อมูลจะอัพเดททุกครั้งที่มีการทำ Test ใหม่
   (ผ่าน PHP server)

หมายเหตุ: วิธีนี้ไม่มีปัญหา CORS!


========================================
วิธีที่ 2: เปิดแบบ Dynamic (ต้องใช้ PHP Server)
========================================

1. เปิด Command Prompt / Terminal

2. ไปที่โฟลเดอร์ C:\Apps\Rehap:
   cd C:\Apps\Rehap

3. รันคำสั่ง:
   php -S localhost:8000

   หรือดับเบิลคลิก: start_server.bat

4. เปิดเบราว์เซอร์ที่:
   http://localhost:8000/pdf/report/dashboard.html

5. เมื่อเสร็จแล้วกด Ctrl+C เพื่อหยุด Server


========================================
การทำงานของระบบ
========================================

1. เมื่อผู้ใช้ทำ Test ใน index.html:
   - ข้อมูลจะถูกส่งไปที่ save_test_data.php
   - บันทึกลง data.json
   - อัพเดท data.js อัตโนมัติ

2. Dashboard จะอ่านข้อมูลจาก:
   - data.js (ถ้าเปิดแบบ Static)
   - get_data.php (ถ้าเปิดแบบ Dynamic)

========================================
การแก้ปัญหา
========================================

Q: Dashboard ไม่แสดงข้อมูล?
A: 1. ตรวจสอบว่ามีไฟล์ data.js หรือไม่
   2. เปิด Console (F12) ดู error
   3. ลองใช้วิธีที่ 2 (PHP Server)

Q: ข้อมูลไม่อัพเดท?
A: 1. ตรวจสอบว่า PHP Server กำลังทำงาน
   2. ลอง Refresh หน้า Dashboard
   3. ตรวจสอบ data.json และ data.js

Q: ไม่มี PHP?
A: ดาวน์โหลดติดตั้ง PHP จาก:
   https://windows.php.net/download/
